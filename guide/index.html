# 概述

欢迎访问 GeckoRelay 文档中心！

这个语言版本的文档是 GeckoRelay 官方人员创建的，最为准确，所有其他语言的翻译版应当以此版本为准。

由于存在大量的客户端，而 GeckoRelay 难以为所有客户端都进行兼容性测试并撰写支持文档，因此我们在此列出目前受到 GeckoRelay 支持或曾经受到 GeckoRelay 支持的客户端以便用户据此购买和使用客户端。

GeckoRelay 会根据客户端的使用人数和其功能性、可靠性来动态地调整支持状态。同时上游代码的更新和客户端的重大更新也可能导致 GeckoRelay 调整其支持状态

主流支持

GeckoRelay 目前提供对此客户端的主流支持

主流支持的客户端其文档页面会显示如上的提示信息。

所有在这个支持级别的客户端，GeckoRelay 都会提供完整的 API / 用户界面支持和功能支持，保持我们关于此客户端的知识库文档始终处于最新。并且在大多数情况下可以在工单中协助用户处理有关此客户端使用问题（标准支持服务及更高级支持服务）。

扩展支持

GeckoRelay 目前提供对此客户端的扩展支持

扩展支持的客户端其文档页面会显示如上的提示信息。

某些客户端因为 GeckoRelay 功能性更新或客户端稳定性问题，就可能不再被 GeckoRelay 推荐使用并转入扩展支持状态。GeckoRelay 仍然会更新其对应的知识库文档，但可能无法保障时效性，并且 API 和用户界面可能不会提供对此客户端的适配。

同时，GeckoRelay 的标准支持服务不会处理有关此类客户端的问题，在此类客户端上使用 GeckoRelay 服务很可能不受到 GeckoRelay SLA 的保障。

即将支持

GeckoRelay 将很快提供此客户端的支持

即将支持的客户端其预览文档页面会显示如上的提示信息。

一般而言，我们仅在「虚拟化」文档中会使用此标签。

但是某些新的优秀客户端可能会被 GeckoRelay 加入快速支持流程而使用此标签。在 GeckoRelay 支持此客户端前，其支持状态应当被视为「从未支持」。

停止支持

GeckoRelay 不再提供对此客户端的支持

停止支持的客户端其文档页面会显示如上的提示信息。

由于客户端软件停止维护或严重的性能或稳定性问题，rixCloud 可能停止对这些客户端的支持，其知识库的文档页面会保留，但不再更新（除非此客户端重新恢复更新或解决了之前的问题）。并且，GeckoRelay API 和用户界面不会提供对此类客户端的支持，虽然 GeckoRelay 服务很可能能够继续在此类客户端上使用，但用户很可能必须手动配置此类客户端。

同时，在此类客户端上使用 GeckoRelay 服务不受到 GeckoRelay SLA 的保障。

从未支持

这些客户端因为其功能性或稳定性问题不会被 GeckoRelay 支持，也可能因为其违反了开源协议或侵犯了他人版权，GeckoRelay 永远不会支持此类客户端。

GeckoRelay 服务可能能够在这些客户端上运行，但是其安全风险需要用户自行承担，某些客户端会上传用户的服务器配置到其服务器，这可能会导致我们暂停或永久限制这些被泄露的订阅和账户。同时，在此类客户端上使用 GeckoRelay 服务不受到 GeckoRelay SLA 的保障。

如果您认为 GeckoRelay 不支持某客户端是错误的，或此客户端已经修正了之前的问题，请通过工单告知我们。
