# SSR C-sharp 配置

# 概述

SSR C-sharp 是 SSR 主分支维护的 Windows 版 SSR 客户端，由 C-sharp（C#）语言编写，对 SSR 协议支持最完善。

然而很遗憾的，SSR 主分支已经停止维护，其分支版本 SSRR 仍有维护，但未有主要更改，GeckoRelay 目前也未有计划采用此分支。

扩展支持

GeckoRelay 目前提供对此客户端的扩展支持

目前正在更新由 GeckoRelay 维护的 SSR C-sharp 分支版本，在此分支释出前，为了避免重复编写文档，暂不提供关于此客户端的配置文档。
